Contributors: pronamic, remcotolsma 
Tags: ipad, redirect
Donate link: http://pronamic.eu/donate/?for=wp-plugin-ipad-redirect&source=wp-plugin-readme-txt
Requires at least: 3.0
Tested up to: 3.4.1
Stable tag: 0.1

This plugin makes it easy to redirect iPad users to an alternative URL.

== Description ==



== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your 
WordPress installation and then activate the Plugin from Plugins page.
